```
User Opens Browser
    ↓
    ├→ frontend/.env (config)
    │
    → frontend/src/index.js (entry)
    ↓
    → frontend/src/App.js (renders)
    ├→ ImageUploader.js (component)
    ├→ RiskPredictor.js (component)
    │
    → User Action (upload/predict)
    ↓
    → axios (HTTP client)
    ↓
    → backend/app.py (Flask server)
    ├→ /predict-image
    │   ├→ image_classifier.py
    │   └→ saved_models/food_classifier.h5
    │
    ├→ /predict-risk
    │   ├→ risk_predictor.py
    │   └→ saved_models/risk_predictor.pkl
    │
    → Response (JSON)
    ↓
    → Component updates
    ↓
    → User sees result
```
